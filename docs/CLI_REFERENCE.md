## Overview

The AI Voice Agent provides a comprehensive CLI with 8main commands:

```bash
transcriber [COMMAND] [OPTIONS]
```

### Available Commands

| Command | Description |
|---------|-------------|
| [`start`](#start) | Start the AI voice agent |
| [`chat`](#chat) | Start text-only chat mode |
| [`list-tools`](#list-tools) | List available tools |
| [`list-sessions`](#list-sessions) | List conversation sessions |
| [`replay`](#replay) | Replay a conversation session |
| [`export`](#export) | Export session to file |
| [`performance`](#performance) | Show performance metrics |
| [`benchmark`](#benchmark) | Run performance benchmarks |
| [`profile`](#profile) | Profile performance |
| [`configure`](#configure) | Interactive configuration |
| [`cleanup`](#cleanup) | Clean up old sessions |

## Core Commands

### `start`

Start the AI voice agent with full audio pipeline.

```bash
transcriber start [OPTIONS]
```

#### Options

| Option | Type | Default | Description |
|--------|------|---------|-------------|
| `--model`, `-m` | `str` | `llama3.2:3b` | Ollama model to use |
| `--device`, `-d` | `int` | `None` | Audio input device index |
| `--list-devices`, `-l` | `flag` | `False` | List available audio devices |

#### Examples

```bash
# Start with default settings
transcriber start

# Use a different model
transcriber start --model llama3.2:1b

# Use specific audio device
transcriber start --device 2

# List available audio devices
transcriber start --list-devices
```

#### Audio Device Output

When using `--list-devices`:

```
Available Audio Devices:

🎤 Input devices (microphones):
  0: Built-in Microphone (2 channels)
  1: USB Headset Microphone (1 channels)
  2: External USB Mic (1 channels)

🔊 Output devices (speakers):
  0: Built-in Speakers (2 channels)
  1: USB Headset Speakers (2 channels)
  2: Bluetooth Headphones (2 channels)
```

### `chat`

Start text-only chat mode for testing and development.

```bash
transcriber chat [OPTIONS]
```

#### Options

| Option | Type | Default | Description |
|--------|------|---------|-------------|
| `--model`, `-m` | `str` | `llama3.2:3b` | Ollama model to use |

#### Examples

```bash
# Start text chat with default model
transcriber chat

# Use smaller model for faster responses
transcriber chat --model llama3.2:1b

# Use larger model for better quality
transcriber chat --model llama3.2:7b
```

#### Interactive Commands

Within chat mode, you can use these commands:

- `exit` or `quit` - Exit chat mode
- `clear` - Clear conversation history
- `help` - Show available commands
- `tools` - List available tools
- `sessions` - Show current session info

## Tool Management

### `list-tools`

List all available tools with descriptions and usage examples.

```bash
transcriber list-tools [OPTIONS]
```

#### Options

| Option | Type | Default | Description |
|--------|------|---------|-------------|
| `--category`, `-c` | `str` | `None` | Filter by tool category |
| `--search`, `-s` | `str` | `None` | Search tools by name or description |
| `--detailed`, `-d` | `flag` | `False` | Show detailed tool information |

#### Categories

- `system` - System operations (files, processes, environment)
- `utility` - Utility functions (calculator, text processing)
- `information` - Information retrieval (search, documentation)
- `productivity` - Productivity tools (notes, tasks, timers)

#### Examples

```bash
# List all tools (summary view)
transcriber list-tools

# Show detailed information for all tools
transcriber list-tools --detailed

# Filter by category
transcriber list-tools --category system
transcriber list-tools --category utility

# Search for specific tools
transcriber list-tools --search "file"
transcriber list-tools --search "calculator"

# Combine filters
transcriber list-tools --category system --search "process"
```
